# Healthy Mind Admin

Admin panel for the Healthy Mind application.

## Setup

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  **Important**: You must add your own `google-services.json` file to the `app/` directory. This file is required for Firebase services to work and is excluded from version control for security.
4.  Sync Gradle and run the app.

## Technologies
- Android SDK
- Firebase
